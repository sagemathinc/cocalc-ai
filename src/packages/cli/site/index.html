CoCalc CLI

The command-line interface for CoCalc project, host, browser, and agent automation. Install the standalone SEA binary with one terminal command.

Linux x64/arm64 macOS arm64 Requires curl + xz

Install

Run this command in a terminal:

curl -fsSL https://software.cocalc.ai/software/cocalc/install.sh | bash

Direct link: install.sh 

$ cocalc --help
CoCalc CLI prints the available automation commands.

What this does

The installer keeps each CLI version isolated and verifiable.

  1. Reads the platform manifest. It selects the current CoCalc CLI artifact for your operating system and CPU architecture.
  2. Verifies the download. The artifact is checked with the SHA-256 digest published in the manifest before it is installed.
  3. Creates a launcher. Linux installs under ~/.local/share/cocalc with ~/.local/bin/cocalc. macOS installs under the same XDG-style locations unless you set COCALC_CLI_HOME or XDG_DATA_HOME.

Uninstall

Remove the install directory and launcher:

rm -rf ~/.local/share/cocalc ~/.local/bin/cocalc

If the installer added PATH modifications, remove the line it appended to your shell rc file, such as ~/.bashrc, ~/.zshrc, or ~/.config/fish/config.fish.
