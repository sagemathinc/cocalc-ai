# Legacy Scripts

These scripts are old top-level operational helpers preserved during the
June 2026 `src/scripts` cleanup.

They had no current repository references when moved here, and many predate the
current Launchpad, Star, bay, Conat, and pnpm workspace layouts. Treat them as
historical source material, not supported commands.

Before using or documenting any script from this directory:

1. Check its last commit and current dependencies.
2. Port it to the current architecture.
3. Move it into a purpose-specific `src/scripts/<area>/` directory.
4. Add README/package-script references and focused validation.
